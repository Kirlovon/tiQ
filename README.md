# tiQ
 Tiny virtual machine written in TypeScript

## Instruction Set _(4 bits)_
0. NOTHING, 4096
1. LOAD, 4096
2. SAVE, 4096

3. ADD, 16, 16, 16
4. SUBSTRACT, 16, 16, 16
5. INCREASE
6. DECREASE

7. EQUAL
8. LESS
9. GREATER

10. AND, 16, 16, 16
11. OR, 16, 16, 16
12. XOR, 16, 16, 16
13. NOT, 16, 16, 16

14. JUMP
15. CLEAN
